# 🏸 ShuttleStats – Badminton Player Explorer

## 📌 Project Description

ShuttleStats is a web application that allows users to explore badminton players, view their rankings, and analyze performance statistics. The application focuses on providing an interactive and user-friendly interface where users can search, filter, and sort player data efficiently.

This project demonstrates the use of JavaScript concepts such as API integration, array higher-order functions, and dynamic UI rendering.

---

## 🎯 Purpose

The goal of this project is to:

* Practice working with APIs using `fetch`
* Implement real-time data handling in JavaScript
* Build an interactive and responsive UI
* Apply array methods like `map`, `filter`, and `sort`

---

## 🔗 API Used

The project will use:

* A public sports API (via RapidAPI) OR
* A local JSON file (simulating API data if required)

The data will include:

* Player name
* Country
* Ranking
* Match statistics
